# AlphaHunt 🔤

A full-stack word hunting game built with React and Node.js.

## Features
- Hunt words starting with every letter of the alphabet
- 470,000+ words from open-source dictionary
- Progress saved per user, per letter
- JWT authentication
- Persistent progress across devices and sessions

## Tech Stack
- React, Tailwind CSS, React Router
- Axios, Vite
